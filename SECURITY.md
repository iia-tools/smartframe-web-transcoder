# Security

## 설계 원칙

- 영상은 GitHub 또는 제3자 서버로 업로드하지 않습니다.
- SmartFrame PIN과 관리자 세션 쿠키를 이 페이지에 전달하지 않습니다.
- 토큰은 암호학적으로 무작위인 256비트 값이며, 선택한 캐시 영상 한 편에만 연결되고 10분 뒤 만료됩니다.
- 토큰으로 가능한 작업은 해당 원본의 읽기, 변환 결과의 순차 업로드, 완료 또는 취소뿐입니다.
- SmartFrame Hub는 허용된 GitHub Pages Origin과 로컬 개발 Origin만 CORS로 허용합니다.
- 업로드는 최대 1GB, 한 조각은 최대 8MB로 제한하며 순서가 어긋나면 거부합니다.
- 최종 파일에서 긴 변 1280px 이하의 H.264 영상 트랙을 확인한 뒤에만 캐시로 가져옵니다.

## 신고

공개 이슈에 PIN, 토큰, 사설 주소, 영상 샘플을 올리지 마세요. 저장소의 GitHub Security Advisory 기능으로 비공개 신고해 주세요.
